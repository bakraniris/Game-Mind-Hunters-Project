# Game-Mind-Hunters-Project

## Server

### Game Startup

```bash
# change working directory
cd server

# Install dependencies
npm install

# Start the server
npm start
```

The server will run at http://localhost:3000

## Project Structure

```

│   ├── index.html

└── server/             # Backend
```
